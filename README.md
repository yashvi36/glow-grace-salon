# Glow & Grace Salon Website

A modern and responsive salon website created to showcase beauty and grooming services in a professional way.

---

## Features
- Responsive design (mobile & desktop)
- Service listing with images and pricing
- Service detail pages
- Clean UI for local businesses
- Easy to customize for salons, spas, or similar services

---

## Built With
- HTML5
- CSS3
- Bootstrap
- JavaScript

---

## Project Purpose
This project is created for portfolio and client demo purposes.  
It helps local salon businesses present their services online and attract customers.
